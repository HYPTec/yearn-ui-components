## Additional Notes
### Recommended
- VsCode
  - bradlc.vscode-tailwindcss extension
  - Add this to your config
    ```
    "tailwindCSS.emmetCompletions": true,
    "tailwindCSS.experimental.classRegex": [
      "tw\\=\"([^`]*)\"",
      "tw\\.[a-z]*`([^`]*)",
      "tw`([^`]*)",
      ["classnames\\(([^)]*)\\)", "'([^']*)'"]
    ],
    ```
